# FHEVM Agent Skill

This repo contains a focused AI-agent skill for building confidential smart contracts with Zama FHEVM.

- `SKILL.md` contains the agent instructions, examples, patterns, and self-checklist.
- `EVAL.md` contains prompts and checks for evaluating whether the skill helps agents avoid common FHEVM mistakes.

The goal is not to provide a full app. The goal is to make AI coding agents generate safer FHEVM code.
